CALL-API - ИНТЕГРАЦИЯ АВТОМАТИЧЕСКОГО ОБЗВОНА В БИЗНЕС-ПРОЦЕССЫ

Технологии Call-API позволяют управлять телефонными звонками из внешних систем: озвучить абоненту текстовое сообщение (Text-to-Speech) или подготовленный аудиофайл, получить ответную информацию и распознать речь (Speech-to-text)

Регистрация

интеграция

Посредством Call-API удобно проводить интеграцию сервиса автоматического обзвона (голосового робота) с:

  • CRM-системами для автоматизации воронки продаж
  • CMS-системами управления сайтом (пример- обработка заказов интернет- магазином)
  • Оперативность- получение актуальных данных не займет большого количества времени.
  • ПО для проведения NPS-опросов различными приложениями для отработки голосовых команд и оповещений

Совершение звонка происходит следующим образом

В запросе указываются параметры автоматического обзвона: номер абонента, АОН, текст сообщения, время начала звонка, количество попыток дозвона, время окончания звонка

По окончанию обзвона сервис возвращает отчет: успешный\неуспешный дозвон, длительность прослушивания сообщения, полученный ответ

Методы Call-API позволяют гибко использовать автоматический обзвон роботом в различных сферах автоматизации бизнеса

Подробная  инструкция по использованию Call-API  находиться в разделе  Помощь.
